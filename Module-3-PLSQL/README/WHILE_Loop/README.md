# PL/SQL WHILE Loop

## Concept Explanation
The `WHILE` loop in PL/SQL executes a block of statements repeatedly as long as a specified condition remains true. The condition is evaluated *before* entering the loop body. If the condition is initially false, the loop body is never executed.
