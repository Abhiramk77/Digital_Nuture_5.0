# Module 5 - Spring Core and Maven

This module covers the fundamentals of Spring Framework and Maven dependency management for Java applications.

## Topics Covered

- Spring Framework Introduction and IoC Container
- Maven Project Setup and Dependency Management
- ApplicationContext and Bean Configuration
- Component Scanning and Annotation-based Configuration
- Dependency Injection with `@Service`, `@Component`, `@Configuration`

## Tools Used

- Java 17
- Spring Framework 6.1.11
- Maven 3.x
- JUnit 5

## Structure

- Exercise 1: Spring Framework Introduction
- Exercise 2: Maven Dependency Management

## Status

Completed

## Learning Outcome

While working on this module, I understood how Spring Framework implements Inversion of Control (IoC) and how Maven simplifies dependency resolution by declaring libraries in `pom.xml` rather than manually managing JAR files.
