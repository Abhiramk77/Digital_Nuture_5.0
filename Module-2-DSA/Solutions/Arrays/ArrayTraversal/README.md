# Array Traversal

## Problem Statement
Perform linear traversal on a given single-dimensional array, demonstrating how to access, read, and print elements sequentially in both forward and backward directions.

---

## Complexity Analysis
- **Time Complexity**:
  - Forward Traversal: $\mathcal{O}(n)$
  - Backward Traversal: $\mathcal{O}(n)$
- **Space Complexity**: $\mathcal{O}(1)$ (Auxiliary space, no extra allocations)

---

## Input and Output Examples

### Sample Input
```text
Array: [12, 35, 9, 56, 23]
```

### Expected Output
```text
=== Array Traversal Demonstration ===
Forward Traversal: 12 35 9 56 23 
Backward Traversal: 23 56 9 35 12 
```
