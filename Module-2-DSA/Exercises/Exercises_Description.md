# Module 2: Data Structures and Algorithms - Exercises

This document outlines the exercises solved under Module 2.

## Array Exercises
1. **Array Traversal**: Implement forward and backward element print.
2. **Linear Search**: Sequentially scan an unsorted array for a target key.
3. **Binary Search**: Perform divide-and-conquer binary search on a pre-sorted array.
4. **Bubble Sort**: Sort an array using adjacent element comparisons and early-exit swaps.
5. **Insertion Sort**: Sort an array by shifting larger values and inserting elements.
6. **Merge Sort**: Recursively partition the array and merge sorted sublists.
7. **Quick Sort**: Pivot-based recursive array partitioning and sorting.
8. **Heap Sort**: Build a binary max-heap to sort array elements.

## Linked List Exercises
9. **Singly Linked List**: Basic pointer chain list with tail insertion, node deletion, and print.
10. **Doubly Linked List**: Bi-directional list with prev/next pointers, tail insertion, deletion, and bi-directional traversals.
11. **Circular Linked List**: List where tail references head to form a loop.
