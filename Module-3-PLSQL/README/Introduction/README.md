# PL/SQL Introduction

## Concept Explanation
PL/SQL (Procedural Language extension to SQL) is Oracle Corporation's proprietary procedural extension for SQL and the Oracle relational database. 

This introductory exercise demonstrates how to configure the client environment to display server-side logs using `SET SERVEROUTPUT ON` and write a simple procedural block executing the standard printing package `DBMS_OUTPUT`.

---

## SQL Execution

```sql

```
