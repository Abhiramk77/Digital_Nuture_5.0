# Submission Guide - Digital Nurture 5.0

## Steps to Submit
1. **Locate folder**: Ensure the folder `DigitalNurture_AbhiRam` contains all four modules.
2. **Zip the folder**: Compress the `DigitalNurture_AbhiRam` folder using your operating system's standard compress feature (e.g. Right Click -> Send to -> Compressed (zipped) folder on Windows).
3. **Verify Upload**: Upload the ZIP file and/or push the code to your GitHub Repository:
   - **Repository Link**: [https://github.com/Abhiramk77/Digital_Nuture_5.0](https://github.com/Abhiramk77/Digital_Nuture_5.0)
4. **Submit in Portal**: Submit the repository URL and ZIP file to the official Digital Nurture portal.
