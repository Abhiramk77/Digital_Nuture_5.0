# Module 6 - Spring Data JPA with Spring Boot and Hibernate

This module covers Spring Data JPA, Hibernate ORM, repositories, entity mapping, pagination, sorting, and REST API development with Spring Boot.

## Developed By

**Name**: Abhiram

## Topics Covered

- Spring Data JPA Repositories (`JpaRepository`)
- Entity Mapping with Hibernate annotations (`@Entity`, `@Table`, `@Id`, `@GeneratedValue`, `@Column`)
- Custom Finder Methods & JPQL Queries (`@Query`)
- Layered Architecture (Controller -> Service -> Repository -> Entity / DTO)
- Request Validation (`@Valid`, `@NotBlank`, `@Email`, etc.)
- Global Exception Handling (`@RestControllerAdvice`, `@ExceptionHandler`)
- SLF4J Logging (`@Slf4j`)
- Pagination & Sorting (`Pageable`, `Sort`)
- Lombok Annotations (`@Getter`, `@Setter`, `@NoArgsConstructor`, `@AllArgsConstructor`)
